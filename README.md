    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1><a id="user-content-getting-and-cleaning-data-project" class="anchor" href="#getting-and-cleaning-data-project" aria-hidden="true"><span class="octicon octicon-link"></span></a>Getting-and-Cleaning-Data-project</h1>

<p>run_analysis.R   Version 1            by John Allen</p>

<p>This script takes wearable computing data from accelerometers from the Samsung Galaxy S smartphone and produces a tidy dataset for further analysis.</p>

<p>This URL has the data's description    <a href="http://archive.ics.uci.edu/ml/datasets/Human+Activity+Recognition+Using+Smartphones">http://archive.ics.uci.edu/ml/datasets/Human+Activity+Recognition+Using+Smartphones</a> </p>

<p>This URL has the original data         <a href="https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip">https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip</a> </p>

<p>run_analysis.R contains the following steps: </p>

<p>Step 1 - Merges the training and the test sets to create one data set.</p>

<p>Step 2 - Extracts only the measurements on the mean and standard deviation for each measurement. </p>

<p>Step 3 - Uses descriptive activity names to name the activities in the data set.</p>

<p>Step 4 - Appropriately labels the data set with descriptive variable names. </p>

<p>Step 5 - From the data set in step 4, creates a second, independent tidy data set with the average of each variable for each    activity and each subject.</p>
